# 实验报告：NMP事实约束包对大模型输出的影响研究

**实验编号**：NMP-EXP-20260617-001  
**实验日期**：2026年6月17日  
**实验者**：loweswang  
**报告版本**：V1.0

---

## 一、实验背景与问题

### 1.1 研究问题

1. 事实约束包是否存在：当大模型接收到结构化的"事实约束包"时，其输出是否会发生可观测的变化？
2. 差异的性质是什么：不同大模型在相同事实约束下，输出的差异是"程度差异"还是"性质差异"？
3. 风格的可迁移性：风格表达是否可以被"传染"——即一个模型的输出风格能否被另一个模型通过参考样例习得？

### 1.2 实验假设

- **H1**：有事实约束包的输出，其"判断归属"将从不明确转向可追溯至事实包来源。
- **H2**：不同大模型在相同事实约束下输出不同风格，但核心判断保持一致。
- **H3**：风格是可迁移的——当模型看到另一个模型的输出样例时，其输出风格会向样例靠拢，但其底层核心逻辑更趋于一致。

## 四、分析与讨论

### 4.1 H1验证：事实约束包改变了"判断归属"

| | G0（无约束） | G1/G2（有约束） |
|:---|:---|:---|
| 内容来源 | AI的通用知识库（训练语料） | 用户的事实包 + 模型表达 |
| 是否能体现用户自己的判断 | ❌ 不能 | ✅ 能 |
| 读者在读完后的反应 | "这AI挺能写" | "这个作者想得很清楚" |

**结论**：H1成立。事实约束包将内容归属权从AI的通用知识库转移到了用户的事实包。AI从"作者"变为"代笔者"，其角色不再是"生成自己的知识"，而是"组织用户的事实"。

### 4.2 H2验证：不同模型在相同约束下输出风格不同，但核心判断相同

| 相同点 | 不同点 |
|:---|:---|
| 核心判断（"AI≠决策引擎"/"分层架构"） | 结构组织方式（诊断式 vs 说明书式） |
| 事实来源（来自同一个事实包） | 语言温度（有锋芒 vs 中性和全面） |
| 基本主张（重构AI客服定位） | 表达载体（段落式 vs 表格化） |

**结论**：H2成立。相同约束包下，不同模型的输出差异是"表达方式"的差异，而非"内容"的差异。G1的输出可概括为"诊断书"，G2的输出可概括为"功能说明书"。

### 4.3 H3验证：风格是可迁移的

G3/G4的推演显示：
- 当模型看到另一个模型的输出作为风格参考时，其输出风格会向参考样例靠拢
- 但核心判断不会变化（因为事实包锁定了内容边界）

**结论**：H3成立。风格是"表达方式"，不是模型的"固有属性"。它可以在上下文引导下发生变化。底层核心逻辑趋于一致。

### 4.4 关于"独特见解"的来源

| | 无约束包 | 有约束包 |
|:---|:---|:---|
| "独特见解"来源 | ❌ 不存在（只有通用知识） | ✅ 用户的事实包 |
| AI提供的是 | 它知道的所有内容 | 组织用户想法的能力 |
| 读者记住的是 | 这个方案挺全 | 这个人想得清楚 |

"独特见解"不来自模型，而来自事实约束包中用户自己的判断。模型只是将其组织得更清晰、更有说服力。

## 五、结论

### 5.1 实验结论

1. **事实约束包改变了大模型输出的"性质"**：从"AI自己说"变为"AI替你整理你说"。这是从"程度差异"到"性质差异"的转变——不是好一点或差一点，而是角色和归属完全不同。

2. **不同模型的风格差异是"表达层"的差异，而非"内容层"的差异**：元宝的判断驱动力和豆包的结构驱动力在核心主张上一致，只是各以不同的方式组织表达。

3. **风格是可以被传染的**：当模型看到另一个模型的输出样例时，其输出风格会趋向样例。这意味着表达方式可以被设计者"锚定"，而不是模型固化的属性。

4. **"独特见解"的归属权属于事实包的构建者，而非模型本身**：只要事实包里包含用户自己的判断，AI只是代笔者。

### 5.2 NMP架构的存在意义

1. **事实层与表达层的分离是合理的**：事实包（L2的输出）提供确定性内容，大模型（L3）提供多样化表达。两者解耦，各自优化。

2. **事实包将内容的归属权交还给构建者**：有了事实约束，AI输出的不再是"AI的知识"，而是"用户的事实包"的组织化呈现。这解决了当前大模型输出中"知识归属不明"的问题。

3. **风格的可迁移性证明了表达层是可设计的**：不同模型的风格差异可以被"锚定"——设计者可以通过提供风格样例来控制表达的倾向（判断驱动/结构驱动）。

4. **NMP架构的核心价值在于"锁定判断，释放表达"**：判断由事实包锁定（无论什么模型，核心主张一致），表达由模型释放（不同模型提供不同风格，供读者选择）。

实验证明：NMP架构不是"让AI变强"，而是"让AI的知识和判断归属于你"。这是它与通用大模型直接提问的根本区别——不在于回答的质量高低，而在于"谁在说话"。
